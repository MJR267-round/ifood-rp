# iFood RP - Plataforma Completa

Plataforma de delivery tipo iFood para roleplay, com sistema de login, cadastro, perfil de usuário, restaurantes, pedidos, chat e painel administrativo.

## 🎯 Funcionalidades (6 Partes)

1. **Sistema de Autenticação** - Login e Cadastro com persistência
2. **Perfil de Usuário** - Dados pessoais, histórico de pedidos, endereços salvos
3. **Listagem de Restaurantes** - Browse de restaurantes com busca e filtros
4. **Gerenciamento de Pedidos** - Carrinho, checkout, rastreamento
5. **Chat/Suporte** - Comunicação com restaurantes e suporte
6. **Painel Administrativo** - Gerenciar usuários, restaurantes, pedidos (admin only)

## 🛠️ Stack Tecnológico

- **Frontend**: HTML5, CSS3, JavaScript (SPA)
- **Backend**: Node.js + Express
- **Banco de Dados**: MongoDB Atlas
- **Autenticação**: JWT
- **Real-time**: Socket.io
- **Deploy**: Vercel (frontend) + Render (backend)
